##Smart Attendance Management System##

A web-based automated attendance tracking system developed using PHP, MySQL, HTML, CSS, JavaScript, and Chart.js.
This project calculates hour-wise attendance, provides branch-wise, class-wise, and college-wise analytics, and includes multiple user roles (Admin, HOD, Class Lecturer, Subject Lecturer).
